# GUIA RAPIDO - Deploy Vercel

## SCRIPTS DISPONIVEIS

1. `deploy.bat` - Mais simples, dois cliques
2. `deploy-simples.ps1` - PowerShell sem emojis

---

## COMO USAR

### Opcao 1: Script BAT (Recomendado)

1. Copie para sua pasta do projeto
2. Dois cliques em `deploy.bat`
3. Siga as instrucoes

OU no CMD:
```
cd C:\Users\marco\Desktop\aapp_jobmatch-main
deploy.bat
```

---

### Opcao 2: PowerShell

```powershell
cd C:\Users\marco\Desktop\aapp_jobmatch-main

# Se der erro, rode primeiro:
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

# Execute:
.\deploy-simples.ps1
```

1. Remove vercel.json problematico
2. Cria novo vercel.json limpo
3. Verifica Vercel CLI
4. Faz deploy automatico

## DEPOIS DO DEPLOY

1. Copie a URL que aparece
   Exemplo: https://jobmatch-abc123.vercel.app

2. Acesse: https://vercel.com/dashboard

3. Clique no projeto

4. Settings -> Environment Variables

5. Adicione:


## GERAR APK

Depois do deploy funcionando:

1. Va em: https://www.pwabuilder.com/
2. Cole sua URL do Vercel
3. Start -> Package For Stores -> Android
4. Configure:
   - Package ID: com.jobmatch.twa
   - Upload keystore: android-app/android.keystore
   - Password: android123

## TROUBLESHOOTING

### Erro: vercel.json
- Delete o arquivo e rode o script novamente

### Erro: Vercel CLI not found
```
npm install -g vercel
```

### Erro: Module not found
```
npm install
```
